# React App IaaC

This is ec2 instance with react code. An application user who can SSH and deploy code on ec2 instance

## Create Infra

```bash
cd react-iaac
terraform plan
terraform apply
```

## Usage

Application User for app deployment
```bash
ssh react-user@IP
```
